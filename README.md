# Network Inspector

Простой учебный Python-скрипт для базовой сетевой диагностики.

## Возможности

Скрипт умеет:
- получать IP-адрес сайта через DNS
- проверять доступность хоста через ping
- проверять открытые порты
- отправлять HTTP-запросы и смотреть ответ сервера

## Для чего этот проект

Этот проект сделан в учебных целях, чтобы потренироваться работать с:
- Python
- socket
- subprocess
- requests
- обработкой ошибок через try/except

## Запуск

Убедитесь, что у вас установлен Python 3.

Запуск:
```bash
python network_inspector.py
```

## Пример использования

После запуска программа показывает меню, где можно выбрать нужный инструмент:
- DNS lookup
- Ping host
- Check ports
- HTTP check

## Структура проекта

- `network_inspector.py` — основной файл программы

## Запуск через Docker

Сборка образа:

```bash
docker build -t network_inspector .
```

Запуск контейнера:

```bash
docker run -it network_inspector
```

Пример использования:

```text
domain: google.com
```

Для выхода из программы:

```text
X
```
